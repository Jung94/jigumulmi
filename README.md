# 지구멀미(Jigumulmi)

비건 정보 큐레이션 플랫폼으로 '지구 속도에 발맞추는 가장 맛있는 한걸음'이란 슬로건을 가지고 서비스 중입니다.

## 링크

https://www.jigumulmi.com/

## 사용 기술 스택

`React.js` `Next.js` `Tanstack Query` `SCSS`

## 인력 구성

FE 1명, BE 1명, 기획 1명, 마케터 1명 / 기여도: 100%
